## O que é este projeto

Plataforma web interna da V4 Company (unidade Carvalho & Co) para centralizar e visualizar dados de tracking de todos os clientes gerenciados por Gabriel. Consolida Meta Ads, Google Ads, GA4 e e-commerce numa base de dados própria, permitindo entender a jornada completa do usuário do clique até a compra.

## Stack tecnológico

- Next.js (App Router) — frontend
- Firebase Firestore — banco de dados
- Firebase Auth (Email/Password) — autenticação
- Tailwind CSS — estilo
- Vercel — deploy
- n8n/Webhook — ingestão de dados
