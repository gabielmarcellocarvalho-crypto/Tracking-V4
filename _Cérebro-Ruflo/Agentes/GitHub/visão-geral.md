# Agentes GitHub

## pr-manager
- **Uso**: Criar, revisar, mergear PRs
- **Quando**: Código pronto para review/merge

## code-review-swarm
- **Uso**: Review distribuído de código com múltiplos perspectivas
- **Quando**: PRs grandes, código crítico, mudanças de arquitetura

## issue-tracker
- **Uso**: Criar, atualizar, fechar issues; triagem
- **Quando**: Bug encontrado, feature request, gerenciar backlog

## release-manager
- **Uso**: Criar releases, changelogs, versionamento
- **Quando**: Sprint finalizado, hotfix, milestone alcançado
